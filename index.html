Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности описывает, как мобильное приложение Финик (далее — Приложение) собирает, использует и защищает данные пользователей.

Используя Приложение, вы соглашаетесь с условиями данной Политики.

2. Какие данные мы собираем

Финансовые данные (вводятся вами):

  • Доходы, расходы, категории трат
  • Финансовые цели и их параметры
  • Кредиты и регулярные платежи
  • Транзакции (суммы, категории, названия)

Технические данные (собираются автоматически):

  • Идентификатор установки (UUID, не привязан к личности)
  • Модель устройства, версия iOS
  • Язык и часовой пояс
  • Анонимные события использования приложения

Мы НЕ собираем:

  • Имена, email, телефоны, адреса
  • Номера банковских счетов или карт
  • Данные геолокации
  • Контакты, фото, календарь
  • Биометрические данные

3. Как мы используем данные

  • Работа приложения — расчёт бюджета, прогнозов, планов достижения целей
  • Синхронизация — сохранение данных на сервере для резервного копирования
  • AI-ассистент — обработка запросов к финансовому агенту (данные передаются LLM-провайдеру для генерации ответов)
  • Аналитика — анонимные события для улучшения продукта
  • Обратная связь — обработка ваших обращений в поддержку

4. Передача данных третьим лицам

Мы передаём данные только в следующих случаях:

  • Серверная инфраструктура — для синхронизации и резервного копирования ваших финансовых данных
  • LLM-провайдеры — при использовании AI-ассистента контекст запроса передаётся провайдеру языковой модели для генерации ответа

Мы не продаём и не передаём данные рекламным сетям, брокерам данных или иным третьим лицам.

5. Хранение и защита

  • Финансовые данные хранятся локально на устройстве в зашифрованной базе данных
  • Синхронизация с сервером происходит по защищённому каналу (HTTPS/TLS)
  • Токены авторизации хранятся в iOS Keychain
  • Идентификация анонимна — на основе UUID установки, без привязки к личности

6. Ваши права

  • Вы можете удалить все данные, удалив приложение с устройства
  • Вы можете запросить удаление серверных данных, написав нам на email ниже
  • Вы можете не использовать AI-ассистента, чтобы данные не передавались LLM-провайдеру

7. Дети

Приложение не предназначено для лиц младше 16 лет. Мы сознательно не собираем данные детей.

8. Изменения политики

Мы можем обновлять данную Политику. Актуальная версия всегда доступна по этому адресу. Продолжая использовать Приложение после изменений, вы принимаете обновлённые условия.

1. Introduction

This Privacy Policy describes how the Finik mobile application (the "App") collects, uses, and protects user data.

By using the App, you agree to the terms of this Policy.

2. Data We Collect

Financial data (provided by you):

  • Income, expenses, spending categories
  • Financial goals and their parameters
  • Loans and recurring payments
  • Transactions (amounts, categories, names)

Technical data (collected automatically):

  • Installation identifier (UUID, not linked to your identity)
  • Device model, iOS version
  • Language and timezone
  • Anonymous app usage events

We do NOT collect:

  • Names, emails, phone numbers, addresses
  • Bank account or card numbers
  • Location data
  • Contacts, photos, calendar
  • Biometric data

3. How We Use Data

  • App functionality — budget calculations, forecasts, goal planning
  • Sync — server backup of your financial data
  • AI assistant — processing queries to the financial agent (data is sent to an LLM provider to generate responses)
  • Analytics — anonymous events to improve the product
  • Feedback — processing your support requests

4. Data Sharing

We share data only in the following cases:

  • Server infrastructure — for syncing and backing up your financial data
  • LLM providers — when using the AI assistant, query context is sent to a language model provider to generate a response

We do not sell or share data with advertising networks, data brokers, or other third parties.

5. Storage and Security

  • Financial data is stored locally on your device in an encrypted database
  • Server sync uses encrypted channels (HTTPS/TLS)
  • Authorization tokens are stored in the iOS Keychain
  • Identification is anonymous — based on installation UUID, not linked to personal identity

6. Your Rights

  • You can delete all data by removing the app from your device
  • You can request server data deletion by contacting us at the email below
  • You can choose not to use the AI assistant to prevent data from being sent to an LLM provider

7. Children

The App is not intended for persons under the age of 16. We do not knowingly collect data from children.

8. Changes to This Policy

We may update this Policy. The current version is always available at this address. By continuing to use the App after changes, you accept the updated terms.
